LabelledSections
==========
	
This sample demonstrates how to create an alphabetically ordered list with labelled sections.

**If you open the solution in Xamarin Studio, it will not be able to load the Windows Phone project;
and if you open the solution in Xamarin Studio under Windows, it will not be able to load the iOS project either.**

LabelledSections is based off of https://github.com/xamarin/monodroid-samples/tree/master/LabelledSections
